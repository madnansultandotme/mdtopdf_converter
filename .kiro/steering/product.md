# Product Overview

A Markdown to PDF converter web application that allows users to convert Markdown documents to PDF with customizable formatting options.

## Core Features

- Markdown editor with live preview
- PDF generation with configurable typography (font, size, line height)
- Page layout customization (size, margins)
- Custom heading, code block, and table styling
- User authentication via OAuth
- Conversion history tracking

## User Roles

- `user`: Standard authenticated user
- `admin`: Administrative access (assigned to owner via `ownerOpenId`)
